
2.12 批量获取充值记录
~~~~~~~~~~~~~~~~~~~~~~~~

:说明: 批量获取充值记录
:接口地址: /api/v2/billing/depositList
:请求方式: GET
:请求参数:

========= ========== ============= ===================================================
Param	    类型        是否必须       说明
app_id	  String	   可选	          商户唯一表示
data      String	   可选	          加密之后的字符串，解密之后的格式如下定义
========= ========== ============= ===================================================

:请求参数data解密之后数据结构:

========= ======= ========== ===================================================
Param     类型     是否必须    说明
time      long    必填	      当前时间戳
charset   String  必填        编码格式，无特殊情况，传参数utf-8
version   String  必填        接口版本号，无特殊情况，传参数v2
ids       String  必填	      多个充值id使用逗号隔开，最多100个id
========= ======= ========== ===================================================
